# Project-5
This project examines the actions of autonomous vehicles (AVs), conventional vehicles, and pedestrians at an urban intersection through a 200-second video from the TGSIM Foggy Bottom Dataset. The objective of this study is to find out if AVs have better performance than other vehicles.
